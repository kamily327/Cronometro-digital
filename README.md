# Cronômetro Digital

Um cronômetro digital simples feito em **JavaScript**, com funções de **Start**, **Pause** e **Reset**.  
O projeto foi criado para treinar lógica de programação e manipulação de tempo no Front-End.

## Funcionalidades

- Iniciar cronômetro (Start)
- Pausar cronômetro (Pause)
- Resetar para 00:00:00 (Reset)
- Incremento automático de segundos, minutos e horas
- Formatação do tempo com zeros à esquerda (ex: 01:05:09)

## Tecnologias

- HTML
- Tailwind CSS
- JavaScript (Vanilla JS)
